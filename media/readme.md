### 在 Apple Silicon 上使用 Stable Diffusion 的推荐方案

#### 追求极致效率与兼容性
尝试使用 mlx-examples​ 项目来运行 Stable Diffusion​ 模型。这是Apple官方优化的方案，与你的硬件契合度最高。
+ https://github.com/ml-explore/mlx-examples?tab=readme-ov-file

#### 注重中文理解与易用性
在 Draw Things​ 应用中搜索并加载 Z-Image-Turbo​ 模型。这个组合能让你用中文轻松描述想法并快速看到成果。

#### 想要完全控制与深度创作
如果你不介意一些技术配置，可以部署 Stable Diffusion WebUI（如 AUTOMATIC1111）。它将为你打开一扇功能无限的大门，但需要一些学习成本。
