## Use your LM Studio Models in Claude Code


#### 设置claude code的模型路径
```shell
# 替换为你的模型路径
# 写入环境变量
export ANTHROPIC_BASE_URL=http://localhost:1234
export ANTHROPIC_AUTH_TOKEN=lmstudio
```

### 切换模型
```shell
claude --model openai/gpt-oss-20b
claude --model qwopus3.5-27b-v3
```
